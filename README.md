# Esperanto_html_dokumentoj

HTML 文書を内容別に整理しています。GitHub Pages で公開しやすいよう、公開側のフォルダ名はローマ字ベースで統一しました。

## フォルダ構成

- `lernolibroj/fujimaki/`: 藤巻氏の中級独習教科書と各課分割版
- `lernolibroj/esperanto-express/`: `Esperanto Express` 会話教本
- `revuoj/revuo-orienta/2025/`: `La Revuo Orienta` 2025年分のHTML
- `revuoj/revuo-orienta/2026/`: `La Revuo Orienta` 2026年分のHTML
- `rondolegado/2026-03/`: 2026年3月輪読会資料のHTML/Markdown
- `legajxoj/historio-kaj-biografioj/`: 運動史・人物・回想
- `legajxoj/eseoj-kaj-artikoloj/`: エッセイ・記事・言語比較
- `legajxoj/kunvenoj-kaj-prelegoj/`: 例会・講演資料
- `legajxoj/vere-aux-fantazie/`: Claude Piron『Vere aŭ Fantazie』全文対訳
- `_originalaj_materialoj/`: 抽出済みPDFと内部対応表の保管用フォルダ
  公開側の導線からは切り離し、内部確認に必要な資料だけをまとめています。
- `esperanto_html_redaktado/`: 旧 `Esperanto_HTML_文書/` にあった補助資料・スクリプト

## 補足

- タイトルは本文冒頭と照合し、明確に説明不足だったものだけ最小限修正しました。
- HTML 本文の内容そのものは分類整理に必要な範囲を除いて変更していません。
